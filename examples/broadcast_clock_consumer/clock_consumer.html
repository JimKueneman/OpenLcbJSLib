Which clocks should this display?

A layout can have several clocks at once — for example a fast clock for trains and a real-time clock for room lights. Tick the clocks you want this display to show. You'll be able to switch between them on the next screen.

What is this?

This page is a clock face — it listens for time, date, year, and run-speed updates from a clock generator on the network and shows them on screen. You can also drive the generator from here: change the time, change the speed, or start and stop the clock from this page.

Predict between updates

The generator only sends a time update once per minute. Turn on prediction to make this display tick the seconds locally between updates so the time always looks live. Turn it off to only show the values that actually arrive.

Control the Clock
Send commands to the generator on the network. The generator confirms each change by broadcasting the new value back, so you'll see your changes show up in the readout above.
"Refresh" asks the generator to re-send everything (running, speed, year, date, time). Useful if the display ever feels out of sync.
