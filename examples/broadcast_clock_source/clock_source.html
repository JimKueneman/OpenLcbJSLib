Which clocks does this generate?

A layout can have several clocks at once — for example a fast clock that runs trains on a tight schedule, and a real-time clock for room lights or sound effects. Tick the clocks this generator drives. You can leave the others for some other generator on the network to handle.

What is this?

This page acts as the master clock for a model railroad layout. It broadcasts the current time, date, year, and run-speed to any clock displays listening on the network. Pick the clocks above, click Connect, then use the buttons on the next screen to run the clock and broadcast updates.

A "full snapshot" is a quick burst that tells listening clocks the running state, speed, year, date, time, and the next minute — everything they need to catch up in one go. Click this once at the end if you've changed several values in a row.
